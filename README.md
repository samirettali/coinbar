# Coinbar

Minimal macOS menu bar app that shows live Binance price feeds

![Coinbar menu bar screenshot](./coinbar.png)

![Coinbar popup screenshot](./popup.png)

## Timezones

This repo also includes `Zonebar`, a separate menu bar app target for tracking multiple IANA timezones.

```bash
swift run Zonebar
```

Zonebar lets you edit tracked timezones from the popup and choose which clocks appear in the menu bar.
